      This alphabetical list of filename extensions contains extensions of notable file formats used by multiple notable applications or services.

      Ext. Description Used by
      3G2[1] Mobile phone video
      3GP[2] Mobile phone video
      3MF[3] 3D manufacturing format 3D builder
      7z[4] A compressed archive file format that supports several different data compression, encryption and pre-processing algorithms 7-zip
      4TH[5] Forth language source code file Forth development systems
